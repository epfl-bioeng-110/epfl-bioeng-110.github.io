À propos

Comment utiliser cette plateforme

📝 Quiz – Testez vos connaissances avec des questions de différents niveaux. Le système de scoring prend en compte l'utilisation des indices (-50%) et les réponses partielles.

🎴 Flashcards – Révisez rapidement les concepts clés.

🚩 Questions marquées – Marquez les questions importantes pendant les quiz pour les retrouver et les réviser plus tard.

⏭️ Passer une question – Si vous tombez sur une question récemment répondue, passez-la sans qu'elle compte dans votre score.

📊 Statistiques – Suivez votre progression. Cliquez sur une question pour la voir en détail.

Système de scoring

  • Réponse correcte : 1 point (0.5 si indice utilisé)
  • Réponse partielle : Points proportionnels aux bonnes réponses
  • Réponse incorrecte : 0 point

💡 Conseils de révision

  • Répétition espacée : Révisez régulièrement plutôt que tout en une fois
  • Testez-vous : Les quiz sont plus efficaces que la relecture passive
  • Marquez vos difficultés : Utilisez le système de marquage pour créer votre propre parcours de révision
  • Variez les modes : Alternez entre quiz et flashcards pour stimuler différentes formes de mémoire
